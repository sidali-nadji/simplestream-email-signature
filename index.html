Adding your signature to Outlook
1
Copy the signature
Click Copy Signature at the top right. This copies the formatted signature — no file needed.
2
Open Outlook Signatures
In Outlook: File → Options → Mail → Signatures…
3
Create a new signature
Click New and type a name (e.g. your name), then click inside the signature editing area.
4
Paste the signature
Press Ctrl + V. The formatted signature will appear directly — no copy from browser required.
5
Set as default
Under Choose default signature, select it for New messages and Replies/forwards, then click OK.

Copy the signature
Click Copy Signature at the top right. This copies the formatted signature — no file needed.
2
Open Outlook Preferences
In Outlook for Mac: Outlook menu → Preferences → Signatures. Click + to add a new signature and give it a name.
3
Paste the signature
Click inside the signature editing area and press Cmd + V. The formatted signature will appear directly.
4
Set as default
Under Default signatures, select it for New messages and Replies & Forwards, then close Preferences.
Tip: If the pasted signature loses its formatting, make sure you are clicking inside the signature edit area before pressing paste — clicking outside and then pasting may paste as plain text.

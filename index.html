Adding your signature to Outlook
1
Copy the signature HTML
Click Copy HTML at the top right of this page.
2
Open the Signatures folder
Press Win + R, type %APPDATA%\Microsoft\Signatures and press Enter.
3
Save as an HTM file
Open Notepad, paste with Ctrl + V, then File → Save As. Set Save as type to All Files, Encoding to UTF-8, and save the file as YourName.htm inside that Signatures folder.
4
Open Outlook Signatures
In Outlook: File → Options → Mail → Signatures…
5
Create and set as default
Click New and type the same name you used for the file (without .htm). Under Choose default signature, pick it for New messages and Replies/forwards, then click OK.

Copy the signature HTML
Click Copy HTML at the top right of this page.
2
Save as a plain HTML file
Open TextEdit, go to Format → Make Plain Text, paste with Cmd + V, then save as signature.htm.
3
Render it in Safari
Double-click signature.htm to open it in Safari. Press Cmd + A to select all, then Cmd + C to copy the rendered signature.
4
Open Outlook Preferences
In Outlook for Mac: Outlook menu → Preferences → Signatures. Click + to add a new signature and give it a name.
5
Paste and assign
Click inside the signature edit area and press Cmd + V. Under Default signatures, select it for New messages and Replies & Forwards, then close Preferences.
